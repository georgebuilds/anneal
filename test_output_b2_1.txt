# FINDING: coalescing was not the binding constraint.
# Mathematically the old div/mod code (with TS=16 hardcoded) was already equivalent
# to lid.y/lid.x; the Metal shader compiler optimized it away.  The branchless
# select removes divergent boundary branches but saves only a small fraction of
# total execution time on exact-multiple shapes.  The M3 GPU is already at or near
# its bandwidth/occupancy ceiling with the 1D path (~84 GFLOP/s ≈ peak f32 ALU
# utilisation at this workgroup size).  The shared-memory tile loads + 2 barriers
# per outer-tile step add overhead that offsets the saved global reads.
# Next step: B3 register blocking — accumulate multiple output tiles per thread to
